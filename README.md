# 设计师作品集网站 - 简奢风格

这是一个采用简奢产品官网风格设计的设计师作品集网站，旨在展示设计师的专业能力与成果。网站采用深色背景配合金色点缀，营造出高端奢华的视觉体验，同时通过流畅的动画和交互效果提升用户体验。

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- GSAP 动画库

## 主要功能与特性

### 1. 视觉设计
- 深色背景配合金色点缀，营造高端奢华氛围
- 质感文字效果，提升标题和关键信息的视觉冲击力
- 精致的卡片设计和阴影效果，增强页面层次感

### 2. 交互体验
- 灵动岛导航，提供直观的页面位置反馈
- 滚动转场效果，创造沉浸式的视觉体验
- 固定与滚动混合布局，展示产品特性和设计理念
- 全屏抽屉式详情面板，展示作品详细信息

### 3. 响应式设计
- 适配桌面端、平板及移动端
- 移动端优化的导航菜单
- 灵活的网格布局，确保内容在不同设备上的最佳展示效果

### 4. 性能优化
- 图片懒加载
- 平滑的动画过渡
- 优化的代码结构，确保页面加载速度

## 页面结构

1. **导航栏**：固定在顶部，包含网站标识和主要导航链接
2. **Hero 区域**：展示设计师名称和简短职业定位
3. **滚动转场效果区域**：通过滚动展示不同的设计作品
4. **作品展示区**：以网格形式展示精选设计作品
5. **固定与滚动混合布局区域**：展示设计理念、工艺与材质、创新与传统
6. **关于我区域**：展示设计师个人信息、专业技能、工作经验和教育背景
7. **联系方式区域**：提供联系表单，方便潜在客户或雇主联系
8. **页脚**：包含网站标识、社交媒体链接和版权信息

## 本地运行

1. 克隆或下载本项目
2. 打开 `index.html` 文件即可在浏览器中查看网站

## 自定义指南

### 替换内容
1. 修改网站标识和导航链接
2. 更新个人信息、工作经验和教育背景
3. 替换作品图片和详细信息
4. 调整联系表单的提交处理逻辑

### 调整样式
1. 修改配色方案：在 `tailwind.config.js` 中调整 `colors` 对象
2. 调整字体：在 `tailwind.config.js` 中调整 `fontFamily` 对象
3. 自定义动画效果：修改 GSAP 动画配置

## 注意事项

- 确保替换所有示例图片和内容，使其符合您的个人品牌和设计风格
- 测试网站在不同设备和浏览器上的兼容性
- 考虑添加实际的表单提交处理逻辑，以便接收用户留言
